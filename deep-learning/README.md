# Deep Learning Fundamentals

A personal learning project covering core concepts in deep learning, neural networks, and kernel methods.

## Topics

- **Deep Learning Fundamentals** — Forward pass, loss calculation, optimization, and backpropagation
- **Neural Networks** — Architecture (input → hidden → output layers), weight adjustment, activation functions
- **Kernel Methods** — High-dimensional mapping, kernel trick, SVMs, non-linear decision boundaries

## Contents

| File | Description |
|------|-------------|
| `plan.md` | Main conceptual notes on deep learning topics |
| `image-materials/` | Visual aids and workflow diagrams |
| `AGENTS.md` | AI agent instructions for this project |

## Learning Approach

- Progressive: Starts with fundamentals, builds to complex concepts
- Visual-aided: Diagrams alongside text explanations
- Summary-driven: Concise summaries before deep dives
- Link-based: Connects related concepts across sections
