# AI Agent Instructions for Deep Learning Learning Project

## Project Overview
This is a **deep learning fundamentals learning project** focused on understanding core concepts in machine learning and neural networks. The project documents key learning materials, concepts, and visual explanations.

## Key Topics Covered
1. **Deep Learning Fundamentals**
   - Forward pass: Input X → Layers → Prediction Y'
   - Loss calculation and optimization
   - Weight updates through backpropagation

2. **Neural Networks**
   - Architecture: Input layer → Hidden layers → Output layer
   - Weight adjustment during training
   - Activation functions and mathematical transformations

3. **Kernel Methods**
   - High-dimensional space mapping for linear separability
   - Kernel Trick for efficient similarity computation
   - Support Vector Machines (SVMs) and non-linear decision boundaries

## Project Structure
- `plan.md` - Main documentation with conceptual notes on deep learning topics
- `image-materials/` - Visual aids (diagrams, workflow illustrations)
- `deep-learning-python(beginner) 17.pdf` - Reference material for beginners

## How to Assist
When helping with this project:
- **Explaining concepts**: Use clear, beginner-friendly language with analogies
- **Clarifying connections**: Link concepts together (e.g., how loss functions connect to optimization)
- **Visual explanations**: Suggest diagrams or structured visualizations for complex ideas
- **Practical examples**: Provide code examples or pseudocode where helpful
- **Expanding sections**: Help organize and expand existing notes with additional context
- **Reviewing accuracy**: Verify conceptual correctness of explanations

## Learning Style
- Progressive: Starts with fundamentals and builds to more complex concepts
- Visual-aided: Includes images and diagrams alongside text
- Summary-driven: Uses concise summaries before diving into details
- Link-based: Connects related concepts across different sections
